Experiências

React

Uso da biblioteca para a criação de interfaces de usuário, estruturação de novos projetos, adição de novas funcionalidades, uma excelente ferramenta na renderização do projeto. Com ela a criação se torna simples e muito bem estruturada, organização de pastas com cada parte que contém o projeto é fundamental para a refatoração do código.

JavaScript

A linguagem da web, o JavaScritp é o principal responsável por implementar efeitos dinâmicas nas páginas. Com o uso de funções, métodos e tratamento de eventos é possível realizar diversas atividades. Uso de ferramentas nas estruturação de seleção para tomada de decisões, cálculos matemáticos, uso do opereadores relacionais e de decisão fazem com que os projetos tenham os aspectos desejados.

Css

O Css é o que há de mais moderno quando o assunto é estilização da Web, juntamente com o HTML ou JSX é possível criar interfaces incrivéis, junto com o compilador Scss é possivel manter cada parte do projeto separadamente e em conjunto com seus respectivos documentos raiz.

Html

O HTML é uma forma de estruturação de dados na página web, como ele pode-se realizar as marcações como títulos, subtitulos, imagens, link e muito mais. Conhecido com uma das principais linguagens de estruturação do conteúdo nas páginas web.

Git & GitHub

Monitoramento e controle de versão dos códigos, ambas as ferramentas são muito fortes no mercado do desenvolvimento, com elas é possível acompanhar as evoluções e falhas do projeto, tendo todo um histórico das contribuições que foram realizadas, bem como identifica quem e quando forma implementasdas as modificações.
