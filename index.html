Experiências

React

Uma das biblioteca para a criação de interfaces de usuário mais utilizadas no mercado tecnológico. Com ela é possível fazer a estruturação de novos projetos, adição de novas funcionalidades e manutenção de códigos independentemente da linguagem usada. É uma excelente ferramenta na renderização do projeto, pois os script podem ser melhor controlados pelo usuário, além da criação se tornar simples e muito bem estruturada. A organização de pastas com cada parte que contém o projeto é fundamental para a refatoração do código e manutenções futuras.

JavaScript

A linguagem mais conhedida da Web, o JavaScritp é o principal responsável por implementar efeitos dinâmicas nas páginas, está sempre trazendo funcionalidades diversas às páginas em html e css. Com o uso de funções, métodos e tratamento de eventos é possível realizar diversas atividades, o uso de ferramentas nas estruturação de seleção para tomada de decisões, cálculos matemáticos, uso do opereadores relacionais e de decisão fazem com que os projetos tenham os aspectos desejados. É uma linguagem que não tem limites para o que se poder fazer com ela, seja pra o front ou para o backend ela é muito útil.

Css

O Css é o que há de mais moderno quando o assunto é estilização da Web, juntamente com o HTML ou JSX é possível criar interfaces incrivéis. Em conjunto com o compilador Sass se torna ainda mais eficiente para trabalhar e refatorar os projetos de acordo com as necessidades, e tudo isso sem ter que ficar percorrendo logas páginas do documento onde os estilos foram aplicados. Atualmente é uma ferramenta muito poderosa, capaz de fazer coisas surpreendentes.

Html

O HTML é uma das forma de estruturação de dados na página web, como ele pode-se realizar as marcações como títulos, subtitulos, imagens, link e muito mais. Conhecido com uma das principais linguagens de estruturação do conteúdo nas páginas web. Muito usada na criação de formulários, tabelas, areas de texto e muito mais.

Git & GitHub

O Git é uma ferramenta de monitoramento e controle de versão dos códigos, em conjutno com a hospedagem do código no Git Hub são muito fortes no mercado do desenvolvimento, com elas é possível acompanhar as evoluções e falhas do projeto, tendo todo um histórico das contribuições que foram realizadas, bem como identificar quem e quando foram implementadas, além de poder fornecer ao usuário a capacidade de retornar a uma parte do código casso ocorra alguma falha. .
